Click the blue chat bubble to open it. You're talking to the real ChatForge backend running at Ashmin's Laptop. If no API_KEY is set in .env, the built-in fallback engine will respond instead.

What's inside

⚡

Streaming responses

Tokens arrive in real time — no spinner, no wait. Uses SSE and LangChain's astream().

🧠

Conversation memory

The bot remembers the entire chat within the session. Powered by LangGraph's MemorySaver checkpointer.

🔒

Session isolation

Every visitor gets a unique UUID on load — two users chatting simultaneously never see each other's history.

🎛️

Zero-config embed

One <script> tag, two attributes. Works on any HTML page — no npm, no build step.

🛡️

Smart fallback

No API key? The rule-based NLP engine activates automatically — the widget is always online.

🎨

Brand-aware styling

Colour, greeting, and quick-reply buttons all come from .env. Shadow DOM ensures zero style conflicts.

Embed snippet

Copy this tag and paste it before the </body> of any page. Replace the attribute values with your own. 

<script
  src="http://localhost:8000/widget.js"
  data-project-id="Ashmin-Office"
  data-widget-key="change-me-to-a-random-secret"
></script>
